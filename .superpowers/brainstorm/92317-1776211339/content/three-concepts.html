3 Concepts for the Distractions Page

Each one handles profiles differently. Pick the one that feels clearest.

A Expandable Cards

Each profile is a collapsible card. Tap to expand and see/edit its contents. Everything on one scrollable page. Simple, familiar, like an FAQ accordion.

Strengths

  • Editing inline can feel cramped

B Tabbed Profiles

Each profile is a tab. Click a tab to see and edit that profile. Clean, focused — you only see one profile at a time. Like browser tabs.

C Drill-Down (Mobile-Style)

Profile list → tap → full-page editor. Like iOS Settings or Apple's Screen Time. Back button returns to the list. Maximum breathing room.

Strengths

  • Maximum space for editing — full width
  • Clean, uncluttered at every level
  • Scales to any number of profiles
  • Familiar iOS/macOS Settings pattern

Tradeoffs

  • Can't see list + detail simultaneously
  • Extra click to get to editing
  • Navigating back and forth can feel slow

My recommendation: A (Expandable Cards)

Most users will have 1-3 profiles. The accordion keeps everything visible and scannable without navigation. It's the simplest to build (no routing) and the easiest to understand. For a settings page you visit occasionally, reducing clicks matters more than reducing scrolling.
