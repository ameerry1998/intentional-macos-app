4. Start Ritual
Shown when a new block begins. User sets intention before enforcement activates.

Summarize key metrics from Q1 data

Work block start ritual

When: Work block begins (deep work or focus hours) Auto-starts in 3 min if no action
  • Green border glow
  • Block type badge + time range
  • Block title + description (if set)
  • "Start" begins enforcement + timer
  • "Edit" opens inline editor (see below)
  • Auto-start countdown (3 min)
  • No enforcement active while ritual is showing
