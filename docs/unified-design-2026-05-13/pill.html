Pill button → follow-on map

  • Drag handle (visible on hover) → moves pill; position persisted to UserDefault
  • Minimize (-) on any mode → minimize to dock (state 20)
  • Click-through on stats area in .timer → opens Today tab (focuses main window)
  • .timer distraction card "Back to Task" → restores last-relevant tab in browser, dismisses card
  • .startRitual "Start now" → exits ritual, transitions to .timer
  • .startRitual "Edit" → opens .startRitualEdit (state 10)
  • .startRitual "Push back 5 min" → schedules a 5-min delay, returns to noPlan or next pill state
  • .startRitual auto-start countdown → 3 min for work blocks, 30s for free blocks
  • .startRitualEdit "Done" with empty title (work block) → state 11 rejection toast
  • .startRitualEdit "Cancel" → return to .startRitual
  • .celebration dots → manual carousel advance
  • .celebration skip link → end celebration, transition to next state (noPlan or next start ritual)
  • .celebration Up Next card "Start now" → begins next session immediately
  • .noPlan quick block (15/30/60 min) → starts immediate quick block, transitions to .timer
  • .noPlan "Plan Day" / "Schedule Now" → opens Plan tab
  • .noPlan "Snooze 30m" → hides pill for 30 min
  • .noPlan dismiss (-) → minimize to dock (state 20)
  • .bedtimeWindDown T-30 "Minimize until T-15" → hides pill, reappears at T-15
  • .bedtimeLocked "Ask Partner" → opens bedtime unlock-request sheet (overlays.html state 19), pill transitions to state 19 "request sent"
