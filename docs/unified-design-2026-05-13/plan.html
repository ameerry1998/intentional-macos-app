Plan button → follow-on map

  • + Goal → State 7 modal (empty)
  • Click monthly card → State 7 modal (populated)
  • Monthly trash icon → State 22 confirmation
  • + Mission → State 8 modal (empty)
  • Click weekly card → State 8 modal (populated)
  • Strictness pill on weekly card → State 16 picker
  • Strictness Strict → softer (in picker) → State 19 partner unlock
  • Cancel pending strictness → State 18 confirmation
  • Weekly trash icon → State 23 confirmation
  • Drag mission to timeline → State 5 (mid-drag) → State 6 (placed)
  • Click timeline block → mini-popover: edit start/end/duration
  • Drag timeline block body → move (autosave)
  • Drag timeline block edge → resize (snap 15-min)
  • Timeline empty-area click → quick-add modal (overlays.html state 5)
  • "Help me plan" toolbar button → State 9 → 10 → 11 → 12
  • Step Skip / Cancel → overlays.html state 21 (discard confirm)
  • Step Next / Back → step navigation
  • Save 3 goals → State 4 (fully planned)
  • Plan/Review toggle (auto Monday) → State 13 weekly review
  • Done / Slipped / Dropped (in review) → marks row, save indicator
  • Slipped → State 20 carry-over prompt on next-week planning
  • Pattern card click → State 15 expanded
  • "Plan this week →" → exits review, enters State 2 (empty week) or State 9 (help me plan)
  • "Skip review" → stamps a "reviewed" receipt, exits to Plan
  • Open today → → Today tab
  • Monthly review · Complete → goal archived with celebration animation
  • Monthly review · Replace → State 7 modal pre-populated for replacement goal
