Frequently asked questions

Does it work with Safari, Arc, and Firefox?

Yes. Intentional works with Chrome, Safari, Arc, Brave, Edge, Firefox, Opera, and Vivaldi. Browsers with the companion extension get full session UI. Browsers without it still get blocking via AppleScript. Desktop apps are monitored natively.

What if I need YouTube for work?

That's exactly the problem Intentional solves. The AI scores each page against your current task. A React tutorial on YouTube during a coding block? Relevant — it stays. A MrBeast compilation? Not relevant — you get nudged back. Same site, different verdicts.

Can I really not turn it off?

With Strict Mode enabled, the app survives Cmd+Q, relaunches after reboot via a watchdog process, and requires a 6-digit code from your accountability partner to disable. Without Strict Mode, you can quit anytime. Your choice.

Is my browsing data private?

Completely. All AI scoring runs on-device using Apple's Foundation Models or a local Qwen3-4B model. Your page titles, URLs, and usage data never leave your Mac. The only network call is an optional anonymous heartbeat.

Does it slow down my Mac?

No. The app uses Apple's on-device AI models which run on the Neural Engine, or a lightweight local LLM as fallback. Tab polling happens every 10 seconds. CPU usage is negligible — you won't notice it's running.

How is this different from Freedom or Cold Turkey?

Freedom and Cold Turkey use blocklists — every site is either allowed or blocked. Intentional uses AI to understand what you're working on and scores each page individually. Plus: earned browse rewards, progressive enforcement (not binary on/off), and an AI schedule builder. It's not a wall. It's a system.
